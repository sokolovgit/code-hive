# @code-hive/nestjs-common

Shared NestJS utilities and common modules for the Code Hive monorepo.

## Usage

```typescript
import {} from /* exports */ '@code-hive/nestjs-common';
```

## Structure

- `src/index.ts` - Main entry point, exports all public APIs
- Add shared decorators, guards, interceptors, filters, etc. here

## Development

```bash
# Build
pnpm build

# Lint
pnpm lint
```
