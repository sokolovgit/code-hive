# @code-hive/types

Shared TypeScript types and interfaces for the Code Hive monorepo.

## Usage

```typescript
import {} from /* types */ '@code-hive/types';
```

## Structure

- `src/index.ts` - Main entry point, exports all public types
- Add shared types, interfaces, enums, etc. here

## Development

```bash
# Build
pnpm build

# Lint
pnpm lint
```
